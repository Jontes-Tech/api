Welcome to the Arch API by Jonte!

This is an API endpoint with urls to download the latest version of Arch Linux. Below is an example url.

https://api.jontes.page/arch?mirror=rackspace.

Pretty neat, eh? The following mirrors are available:

rackspace, acc-umu and torrent.

To use the redirect option, add this to the end of your url.

&redirect=true

To use the "version-only" version, use this url. All other http queries will be ignored.

https://api.jontes.page/arch?version=true
